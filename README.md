# Flock
---
A sentiment analyzer for the twitter-verse. Enter a search term and see how people on twitter are feeling about that thing, and how their feelings about it have changed over time.
## Compile
npm run react-dev
## Run
npm run server-dev
### Built With:
- React
- Node.js
- Express.js
- Mongoose
- Twitter API
- Sass
